# Paradise Nursery - Shopping Cart Application

## Project Description
Paradise Nursery is a dynamic web e-commerce application built with React and Redux. It allows users to browse a variety of beautiful houseplants, view plant details, add them to a shopping cart, and manage their cart items.

## Features
- **Landing Page:** Welcoming interface with a background image and company details.
- **Product Listing:** Displays 18 unique houseplants categorized into 3 different sections (Air Purifying, Succulents, and Flowering).
- **Shopping Cart Management:** Add items, update quantities, and remove items dynamically.
- **State Management:** Utilizes Redux Toolkit for seamless cart state management.

## Technologies Used
- React.js
- Redux Toolkit (React-Redux)
- CSS for styling
